Cross-Origin Hub Demos

テキスト転送デモ

左と右の入力をリアルタイムに同期します。

  • Left: /demo/left/（ローカルサーバーでは http://127.0.0.1:3000）
  • Right: /demo/right/（ローカルサーバーでは http://127.0.0.1:4000）

WAV送受信デモ

左で生成したサイン波WAVをHub経由で右に送信し、右で再生します。

  • WAV Left: /demo/wav-left/（ローカルサーバーでは http://127.0.0.1:3100）
  • WAV Right: /demo/wav-right/（ローカルサーバーでは http://127.0.0.1:4100）

起動手順

  1. npm install
  2. npm run start でHubサーバーを起動
  3. npm run demo でLeft/Right/WAVデモのローカルサーバーを起動
  4. ブラウザを2つ開き、LeftとRightを別ポートで表示
